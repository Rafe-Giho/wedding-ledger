# Security

## 적용 범위

이 앱은 단일 맥북에서 오프라인으로 사용하는 현장용 장부다. 서버, 계정, 클라우드 동기화는 사용하지 않는다.

## 기본 보안

- 앱 실행 시 비밀번호 필요
- 비밀번호 원문 저장 금지
- 한글/영문 입력 상태가 달라도 같은 물리 키 입력은 같은 비밀번호로 보정
- PBKDF2-SHA256 해시 저장
- 최초 설정 시 복구키 생성
- 복구키 원문 저장 금지
- 복구키로만 비밀번호 재설정 가능
- 일정 시간 미사용 시 자동 잠금
- DB 파일 권한을 사용자 전용으로 제한
- 실제 DB, 백업, 엑셀 파일 Git 제외

## 복구 정책

- 비밀번호를 잊으면 복구키로 새 비밀번호를 설정한다.
- 복구키를 잃으면 앱 비밀번호 복구는 불가능하다.
- 데이터 복구는 백업 파일을 통해 수행한다.
- 백업 복원 전 현재 DB를 자동 백업한다.

## 주의

외부 의존성 없는 가벼운 구현을 우선하므로 DB 파일 자체 암호화는 MVP에서 제외한다. 맥북 사용자 계정, 파일 권한, 앱 비밀번호, 백업 관리로 기본 보호를 제공한다. 더 높은 보안이 필요하면 이후 SQLCipher 또는 OS Keychain 기반 암호화를 추가한다.
