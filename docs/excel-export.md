# Excel Export

## 파일 형식

외부 라이브러리 없이 Excel에서 열 수 있도록 SpreadsheetML 기반 `.xls` 파일을 생성한다.

## 시트

- 전체내역
- 요약
- 모임별
- 검색용
- 수정이력

## 전체내역

모든 원본 기록을 포함한다.

컬럼:

- 봉투번호
- 이름
- 모임
- 관계
- 금액
- 식권수
- 입금방식
- 상태
- 모드
- 입력시간
- 수정시간
- 메모

## 요약

- 정상 기록 수
- 취소 기록 수
- 총 축의금
- 총 식권 수
- 현금 합계
- 계좌 합계
- 기타 합계

## 검색용

Excel에서 직접 조건을 넣어 확인할 수 있는 영역을 제공한다.

- 이름별 건수: `COUNTIF`
- 모임별 총액: `SUMIF`
- 모임별 식권 수: `SUMIF`
- 금액 범위 총액: `SUMIFS`

## 백업

엑셀 추출 전 자동 백업을 생성한다.
