# Excel Export

## 파일 형식

앱은 Excel에서 바로 열 수 있는 `.xlsx` 파일을 생성한다. 외부 Swift 라이브러리는 추가하지 않고, 앱에서 XLSX XML 패키지를 만든 뒤 macOS 기본 zip 도구로 묶는다.

## 시트

- 전체내역
- 요약
- 검색용
- 모임별
- 시간대별
- 동명이인
- 수정이력
- 안내

## 전체내역

모든 원본 기록을 포함한다. 자동 필터가 켜져 있으며 입력시간은 초 단위까지 표시된다.

컬럼:

- 봉투번호
- 이름
- 모임
- 관계
- 금액
- 식권수
- 입금방식
- 상태
- 모드
- 입력시간
- 입력일시값
- 수정시간
- 입력일
- 시간대
- 메모

`입력일시값`은 검색과 시간대별 정산에 쓰는 보조 계산 컬럼이며 엑셀에서는 숨김 처리한다.

## 요약

전체내역을 참조하는 함수로 계산한다.

- 정상 기록 수: `COUNTIF`
- 취소 기록 수: `COUNTIF`
- 총 봉투수: `COUNTIF`
- 총 축의금: `SUMIF`
- 평균 축의금: `AVERAGEIFS`
- 총 식권 수: `SUMIF`
- 결제수단별 합계: `SUMIFS`
- 최초/최근 입력시간: `MINIFS`, `MAXIFS`, `TEXT`

## 검색용

B열에 조건을 입력하면 A16부터 결과가 자동으로 펼쳐진다.

지원 조건:

- 이름 포함
- 모임 포함
- 관계 포함
- 입금방식
- 상태
- 최소 금액
- 최대 금액
- 식권수
- 시작 입력시간
- 종료 입력시간

주요 함수:

- `FILTER`
- `SEARCH`
- `INDEX`
- `ROWS`
- `SUM`
- `DATEVALUE`
- `TIMEVALUE`

입금방식과 상태는 드롭다운으로 선택할 수 있다.

## 모임별

모임별 건수, 총액, 식권 수, 평균 축의금을 계산한다.

주요 함수:

- `COUNTIFS`
- `SUMIFS`
- `IFERROR`

## 시간대별

입력시간 기준 0시부터 23시까지 시간대별 건수, 총액, 식권 수를 계산한다.

주요 함수:

- `COUNTIFS`
- `SUMIFS`

## 동명이인

추출 시점의 정상 기록 중 같은 이름을 찾고, 인원 수, 총액, 식권 수, 봉투번호 목록을 보여준다.

주요 함수:

- `FILTER`
- `COUNTIFS`
- `SUMIFS`
- `TEXTJOIN`

## 수정이력

취소, 복구 등 감사 로그를 보관한다.

## 백업

엑셀 추출 전 자동 백업을 생성한다.
