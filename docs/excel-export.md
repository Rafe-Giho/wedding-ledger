# Excel Export

## 파일 형식

앱은 Excel에서 바로 열 수 있는 `.xlsx` 파일을 생성한다. 외부 Swift 라이브러리는 추가하지 않고, 앱에서 XLSX XML 패키지를 만든 뒤 macOS 기본 zip 도구로 묶는다.

## 시트

- 전체내역
- 요약
- 검색용
- 봉투 검수
- 현금·계좌 정산
- 식권 검수
- 동명이인 점검
- 고액·특이금액
- 취소·삭제 이력
- 안내

## 전체내역

모든 원본 기록을 포함한다. 자동 필터가 켜져 있으며 입력시간은 초 단위까지 표시된다.

컬럼:

- 봉투번호
- 계좌차번
- 이름
- 모임
- 관계
- 대상
- 금액
- 성인식권수
- 소인식권수
- 총식권수
- 입금방식
- 상태
- 모드
- 입력시간
- 입력일시값
- 수정시간
- 입력일
- 메모

`입력일시값`은 검색과 날짜 계산에 쓰는 보조 계산 컬럼이며 엑셀에서는 숨김 처리한다.

## 요약

전체내역을 참조하는 함수로 계산한다.

- 정상/취소 기록 수
- 총 봉투수, 예상 봉투수, 봉투 차이
- 총 축의금, 평균 축의금
- 총 식권 수, 성인/소인 식권 수, 준비 성인/소인 식권 수, 남은 성인/소인 식권 수
- 현금, 계좌, 기타 합계
- 최초/최근 입력시간

## 검색용

B열에 조건을 입력하면 18행부터 검색 결과가 자동으로 채워진다. 시트 안에 사용법과 입력 예시를 함께 표시한다.

지원 조건:

- 이름 포함
- 모임 포함
- 관계 포함
- 대상 포함
- 입금방식
- 상태
- 최소 금액
- 최대 금액
- 성인 식권수
- 소인 식권수
- 시작 입력시간
- 종료 입력시간

입금방식과 상태는 드롭다운으로 선택할 수 있다. 금액과 성인/소인 식권수는 숫자만 입력하는 것을 권장한다. 검색 수식은 Google Sheets와 Excel 365에서 열기 쉬운 `FILTER`, `SUMPRODUCT` 중심으로 구성한다.

## 봉투 검수

마감 때 봉투 실물과 앱 기록을 대조하기 위한 시트다.

- 정상 봉투수
- 취소 봉투수
- 삭제 이력 수
- 누락 봉투번호
- 중복 봉투번호
- 중복 계좌차번
- 취소 기록 목록
- 삭제 이력 목록

## 현금·계좌 정산

입금방식별 금액을 검수한다.

- 현금 정상 건수와 합계
- 계좌 정상 건수와 합계
- 기타 정상 건수와 합계
- 취소 건수
- 실제 현금 입력칸
- 앱 현금 합계와 실제 현금 차이

## 식권 검수

식권 배부와 남은 식권을 확인한다.

- 준비 성인/소인 식권
- 배부 성인/소인 식권
- 남은 성인/소인 식권
- 취소 기록 식권
- 식권 배부 기록 목록

## 동명이인 점검

같은 이름의 정상 기록을 목록으로 보여준다.

- 이름
- 인원
- 차번
- 모임
- 관계
- 대상
- 금액
- 성인식권수
- 소인식권수
- 입금방식
- 입력시간
- 메모

## 고액·특이금액

마감 때 한 번 더 확인하면 좋은 금액을 모아 보여준다.

- 30만원 이상
- 50만원 이상
- 100만원 이상
- 빠른선택 외 금액
- 만원 단위가 아닌 금액
- 0원 기록

## 취소·삭제 이력

잘못 입력한 기록을 추적하기 위한 감사 로그 시트다. 취소, 복구, 삭제 이력을 표시하고 삭제 전 값도 보관한다.

## 안내

각 시트의 목적과 행사명, 총 성인/소인 식권수, 예상 봉투수, 운영 메모를 함께 제공한다.

## 백업

엑셀 추출 전 자동 백업을 생성한다.
