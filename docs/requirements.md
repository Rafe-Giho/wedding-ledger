# Requirements

## 목표

한국 결혼식 축의대에서 한 명의 입력자가 축의금과 식권 수를 빠르게 기록하고, 행사 후 검색/정산/엑셀 추출을 할 수 있는 가벼운 오프라인 프로그램을 만든다.

## 운영 조건

- 입력자는 1명
- 한쪽 축의대 기준으로 운영
- 맥북에어에서 실행
- 인터넷 없이 사용 가능
- GitHub에는 코드만 업로드
- 실제 DB, 백업, 엑셀 파일은 업로드 금지

## 필수 기능

- 앱 비밀번호 설정 및 로그인
- 복구키 기반 비밀번호 재설정
- 축의금 입력
- 봉투번호 자동 증가
- 식권 수 기록
- 빠른 금액 버튼과 직접 금액 입력
- 이름, 모임, 금액, 식권 수, 입금방식 검색
- 총액, 봉투 수, 식권 수, 현금 합계 정산
- 모임별 합계
- 봉투번호 누락 확인
- 동명이인 경고
- 수정 이력 저장
- 삭제 대신 취소 처리
- 테스트 모드와 운영 모드 분리
- 테스트 데이터 초기화
- 자동 백업
- 백업 복원
- 함수/필터/검색 시트를 포함한 `.xlsx` 엑셀 추출

## 필수 입력 항목

| 항목 | 방식 | 설명 |
| --- | --- | --- |
| 봉투번호 | 자동 필수 | 자동 증가, 필요 시 수정 가능 |
| 이름 | 직접 입력 필수 | 검색/정산 핵심 값 |
| 금액 | 필수 | 빠른 버튼 또는 직접 입력 |
| 식권 수 | 필수 | 기본값 0 |
| 입금방식 | 필수 | 기본값 현금 |
| 입력시간 | 자동 필수 | 저장 시 현재 시간을 초 단위까지 자동 기록 |
| 상태 | 자동 필수 | 기본값 정상 |

## 선택 입력 항목

| 항목 | 설명 |
| --- | --- |
| 모임 | 기본값 미분류 |
| 관계 | 가족, 친구, 회사동료 등 |
| 메모 | 특이사항 |
| 수정 사유 | 수정/취소 시 입력 |

## 현장 고려사항

- 큰 글씨와 키보드 중심 입력
- 저장 후 다음 봉투번호 자동 준비
- 금액 0원/음수 차단
- 만원 단위가 아닌 금액은 확인
- 봉투번호 중복 차단
- 동명이인 경고
- 실제 봉투 수와 기록 수 비교
- 실제 현금과 앱 현금 합계 비교
- 총 식권 수 확인
- 행사 전 테스트 입력 후 초기화
- 운영 모드에서는 초기화 방지
- 엑셀 추출 전 자동 백업
- 입력/검색/정산/엑셀에서 입력시간 확인
- 앱 종료 시 자동 백업
